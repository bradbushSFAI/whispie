## Monetization

- **Free tier**: 3 scenarios/month, basic personas, limited analysis, **text only**
- **Pro ($12/mo)**: Unlimited scenarios, all personas, Upload Your Boss, career journal, full analysis, **4 voice chats/month**
- **Voice Pack**: 7 voice chats for $10 (add-on, never expires)
- **Lifetime ($99)**: One-time, all Pro features forever, 4 voice chats/month
- **Team/Enterprise**: Admin dashboard, custom scenarios, progress tracking, bulk voice allocation

### Voice Economics
- ElevenLabs: ~$0.30/1K characters
- Avg conversation: ~2-3K chars = ~$0.60-0.90/chat
- 4 chats included = ~$2.40-3.60 cost on $12 sub (good margin)
- Voice pack (7/$10) = $1.43/chat to user, ~$0.80 cost = healthy margin
- Voice is a **profit center**, not just a feature

## Tech Stack

- **Frontend**: React (Next.js) + Tailwind - **PWA, mobile-first**
- **AI**: Gemini 1.5 Flash (low latency chat)
- **Voice Input**: Web Speech API (free, browser-native)
- **Voice Output**: ElevenLabs (user-selectable voice library)
- **Backend**: Supabase (auth, database, realtime)
- **Payments**: Stripe
- **Analytics**: Posthog free tier (keep it simple)
- **Hosting**: Vercel
- **Moderation**: AI flagging + human override
